## Required

1. Git
   - https://git-scm.com/downloads
2. MySQL
   - https://www.mysql.com/downloads/
3. Node.Js
   - https://nodejs.org/en/download/
4. npm
   - `npm i -g @nestjs/cli`
   - `npm i -g typeorm`
   - `npm install -g ts-node`
